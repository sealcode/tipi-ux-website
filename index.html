Wielki Wodzu UX witamy w Tipi UX!

Tipi UX to regularne spotkania o tematyce User Experience, grafice, psychologii, programowaniu, marketingu czyli o wszystkim tym co jest istotne przy tworzeniu użytecznych rozwiązań.

O co chodzi z tym Tipi?

Tipi UX (skrót od UX tip - wskazówka UX'owa) są spotkaniami stworzonymi dla osób, które dopiero zaczynają swoją przygodę z User Experience oraz dla tych, którzy chcą dalej pogłębiać wiedzę w tym zakresie. Tipi UX stawia głównie na intergracje społeczności UX jak i promocję osób, które chcą dzielić się wiedzą z innymi. Spotkania są w pełni darmowe jednak wymagana jest wcześniejsza rejestracja. Obecnie Tipi UX odbywa się w trzech miastach w Polsce - Warszawie, Poznaniu i w Trójmieście.
